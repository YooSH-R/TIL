1.  
- AI, ML, DL의 정의
AI: 주어진 환경/데이터를 인지/학습/추론을 통해 목표 달성을 하도록 예측/행동 선택/계획 하는 시스템
ML: AI범주 내에서 데이터로부터 학습하여 목적을 달성하는 접근 방법론
DL: ML범주 냄에서 신경망 함수를 사용한 학습 방법론

- 데이터 구성 요소
Feature: 모델이 예측에 사용하는 입력정보, 예측/판단의 근거/단서
Label: 모델이 예측하려는 정답, 학습의 목표값

- 회귀(Regression)
지도학습의 종류, 예측하고 싶은 결과값이 숫자일 때 사용
라벨 및 예측 모델의 출력은 연속적인 수치

- 회귀 설명력: R^2(결정계수)
회귀 모델의 예측 정확도 평가
Label의 분산 중에서 Feature로 설명되는 비율
평균만 쓰는 단순한 예측보다 모델이 실제값을 얼마나 더 잘 맞추는지 0~1 사이 값으로 나타냄
1에 가까울수록 설명력 높고 낮을수록 설명력 낮음
예측값들이 평균값보다 못한다면 음수 나올 수 있음

- 혼동행렬(Confusion Matrix)
분류 모델의 예측 정확도 평가
예측과 실제 값 사이의 관계를 형렬 형태로 표현
TP: 실제 양성, 예측도 양성
TN: 실제 음성, 예측도 음성
FP: 실제 음성, 예측은 양성(오탐)
FN: 실제 양성, 예측은 음성(누락)
정밀도(Precision): 양성이라 판정한 것 중 진짜 양성 비율: TP / (TP + NP)
재현율(Sensitivity): 진짜 양성 가운데 잡아낸 예측 양성 비율: TP / (TP + FN)

2. 
- RSS 뜻과 정의
실제 관측값과 예측값의 차이를 제곱해 합한 값(잔차제곱합)
이 값을 최소화하는게 최소제곱법(least squares)
최소제곱법의 목표: 데이터를 가장 잘 설명하는 직선 찾기 위해 직선의 절편과 기울기 찾기

- 로지스틱 회귀
분류 문제에서 선형회귀의 대안
시그모이드 함수를 활용해 0~1 범위 내 확률값 예측 보장
순서가 없는 범주를 확률로 직접 예측하는 적절한 분류 방법
선형 모형식과 시그모이드 함수의 결합
로짓 변환시 선형 회귀 모형식으로 표현 가능
우도: 확률함수가 데이터를 얼마나 잘 설명하는지 나타낸 짚
MLE(최대 우도 추정): 우도가 최대가 되는 모수를 찾는 과정

- 신경망
입력을 받아 어떤 패턴 학습하고 결과 예측하는 수학적 모델
3가지 layer로 이루어짐: 입력층, 은닉층, 출력층
은닉층 개수에 따라 Shallow 네트워크, Deep 네트워크

- 손실 함수
모델이 예측을 얼마나 잘못했는지 측정하는 함수
학습을 통해 손실 함수를 최소화하는 파라미터(모델의 가중치)를 찾는 과정

- 경사하강법
손실 함수의 값이 줄어드는 방향으로 파라미터를 이동하는 과정
확률적경사하강법(SGD): 전체 데이터를 한번에 쓰는 대신, 무작위로 선택한 데이터 샘플을 사용하여 지역 최소점에 빠지는 문제 해결, 계산량 줄임


3. 
- RNN 원리 이해
가변 길이의 입력을 받을 수 있고, 이전 입력을 기억할 수 있기 때문에 순차적 데이터 처리에 적합한 아키텍처
전통적인 신경망과 달리 이전 시점 정보 담는 hidden state 가지고 있다
따라서, 입력 시퀀스 벡터를 처리할 때 각 시점마다 recurrence 수식을 적용하여 hidden state를 업데이트한다

- LSTM 원리 이해
RNN의 기울기 소실 문제를 해결하기 위해 제안된 RNN의 한 종류
시점 t에 RNN은 길이가 n인 벡터 hidden state와 cell state를 갖는다. 이 때 hidden state는 short-term info.를 cell state는 long-term info.를 저장. LSTM은 cell state에서 3가지 게이트를 통해 정보를 읽고 지우고 기록할 수 있다.

- Transformer 원리 이해
RNN 한계(장기 의존성, 병렬화 불가) 극복하는 Attention(value집합으로부터 필요한 정보를 직접 불러와 결합) 메커니즘.
Attention 메커니즘을 encoder-decoder 간이 아닌, 한 문장 내부에서 적용
Self-Attention(Positional Encoding, Feed-Forward Network, Masked Self-Attention)을 핵심 메커니즘으로 하는 신경망 구조
encoder-decoder 구조로 설계된 신경망 모델
Multi Headed Attention

- BERT 원리 이해
Encoder 모델
양방향 문맥 모두 활용
사전 학습을 위해 입력 단어의 일부를 마스크 토큰으로 치환해, 모델이 이 자리에 올 단어를 예측하도록 학습하는 방법 사용, 문장 간 관계를 학습하여 문맥적 추론 및 문장 수준 이해 태스크에 도움 되도록 설계

- NLP 성능 평가
Natural Language Processing 자연어 처리


4. 
- 디코딩 알고리즘
다음 단어를 선택하는 방법
Greedy Decoding: 가장 확률 높은 다음 토큰 선택
Beam Search: 확률 높은 k개 후보 동시 고려
Sampling: LLM이 제공한 확률 기준으로 랜덤 생성
Sampling with Temperature: T를 통해 LLM이 생성한 확률 분포 임의 조작
Top-K Sampling: 확률 높은 K개 토큰 중에서만 sampling
Top-P sampling: K 고정하는 대신 누적 확률 P에 집중하여 K 자동 조절

- 선호 학습(Preference Learning)
주어진 입력에 대해 적절한 '하나'의 응답이 있다고 가정하는 지시 학습의 한계
다양한 응답 중 사람이 더 선호하는 응답 생성하도록 추가학습

- 거대 언어 모델의 평가
많은 테스크에서의 성능 종합적으로 판단
입력 프롬프트에 따라 예측 바뀌므로 고려
정답 정해진 경우 - 정확도 측정
정답 없는 경우 - 사람이 임의의 정답 작성하고 비교, 텍스트 자체 품질만 측정, 상대적 선호 평가
한계: 위치 편향, 길이 편향, 자기선호편향

5. 
- CNN 원리 이해
합성곱 레이어: 입력 이미지를 필터와 연산하여 특징 맵을 뽑아내는 모듈
컨볼루션: 필터를 이미지 상에서 이동시키면서 내적을 반복 수행, 내적으로 구한 모든 값을 출력 제공

- CNN 연산
3차원 구조를 그대로 보존하면서 연산

- Pooling
CNN 레이어의 출력을 줄여 연산 효율성을 확보, 위치 변화 강건성 확보(입력 내 객체 위치가 다소 변해도 동일한 출력 제공)

- 스트라이드 합성곱
필터를 스트라이드 값만큼 이동한 후 출력 연산
커널을 동시에 학습해서 학습 상수 없는 풀링과 다름
해상도 저하로 인한 정보 손실 적고 풀링 + 합성곱을 하나의 레이어로 대체
