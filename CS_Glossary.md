# Computer Science Glossary
## 소스코드
- 사람이 읽고 작성한 프로그램의 원래 코드, 프로그램이 만들어지는 출발점(source)
- 컴퓨터가 실행하는 코드는 0과 1로 이루어진 실행 코드

## 중앙집중서버 vs 분산서버
- 중앙집중 서버가 관리는 쉽지만 서버 한 번 죽으면 전체 서비스 마비, 트래픽 급증에 취약 등의 약점

## temp
- temporary variable : 임시 변수로 많이 사용

## >>>
- REPL 프롬프트
-REPL = Read – Eval – Print – Loop
- 입력 읽고 - 실행하고 - 결과 출력하고 - 다시 대기

## Scope
- 범위
- Global scope: 코드 어디에서든 참조할 수 있는 공간
- Local scope: 함수가 만든 scope

## 에러코드
- interpolation
- concatenate: 이어 붙이기
- indices: index 복수형
- literal: 글자 그대로의 값 ex) 10 # 정수 리터럴
