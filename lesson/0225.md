# CSS Layout
- 각 요소의 위치와  크기를 조정하여 웹 페이지의 디자인을 결정하는 것

## A. CSS Box Model

### display 속성(박스의 화면 배치 방식)
- 박스 타입 종류
    1. Block 타입
    2. Inline 타입

### 기타 display 속성
1. inline-block
2. none
3. flex

## B. CSS position
- 요소를 Normal Flow에서 제거하여 다른 위치로 배치하는 것
- Position 유형
    1. static
    2. relative
    3. absolute
    4. fixed
    5. sticky
- z-index

## C. CSS Flexbox
- 요소를 행과 열 형태로 배치하는 1차원 레이아웃 방식
- Flexbox 구성 요소
    - main axis
    - cross axis
    - flex container
    - flex item
- Flexbox 속성 목록
    1. Flex Container 관련 속성
        - display
        - flex-direction
        - flex-wrap
        - justify-content
        - align-items
        - align-content
    2. Flex item 관련 속성
        - align-self
        - flex-grow
            - 남는 행 여백을 비율에 따라 각 flex item에 분배
        - flex-basis
        - order
