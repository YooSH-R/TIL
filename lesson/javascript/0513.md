# 변수
- let: 재할당 O, 재선언 X
- const: 재할당 X, 재선언 X
- var: 재할당 O, 재선언 O
- 호이스팅: 코드가 실행되기 전, 변수/함수를 최상단으로 끌어올려 선언하는 것
  - TDZ: 최상단 선언부터 해당 변수 초기화 지점까지의 영역 -> 접근하면 에러

# DOM
- 웹을 구조화된 객체로 제공
- 객체를 프로그래밍 언어로 이용해서 조작 가능
- document 객체: 최상위 객체, 모든 컨텐츠 진입점

# 동적인 웹페이지 컨트롤
1. 선택
  - document.querySelector()
  - document.querySelectorAll()

2. 조작
  - 속성 조작
    - 클래스 조작
      - classList.add()
      - classList.remove()
      - classList.toggle()
    - 일반 속성 조작
      - getAttribute()
      - setAttribute(name, value)
      - removeAttribute()
  - 컨텐츠 조작
    - 요소.textContent
  - DOM 요소를 생성하고, 추가, 삭제
    - document.createElement()
    - Node.appendChild()
    - Node.removeChild()
  - 스타일 조작
