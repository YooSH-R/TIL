## Model Fields 심화
### choices 속성, 실습
- 모델 필드에 미리 정해진 선택지를 부여하는 속성
- help_text: admin이나 form에서 해당 필드 옆에 간단한 안내문으로 표시되는 설명
- verbose_name: admin이나 form에서 필드의 라벨로 표시될 텍스트
- priority_choices: DB에는 정수 저장, admin site나 django form에서는 낮음, 보통, 높음으로 표시됨

```py
# todos/models.py
class Todo(models.Model):
    STATUS_CHOICES = [
        ('TODO', '할 일'),
        ('DOING', '진행 중'),
        ('DONE', '완료'),
    ]
    status = models.CharField(
        max_length=5,
        choices=STATUS_CHOICES,
        default='TODO',
        help_txt='현재 작업 상태를 선택해주세요.',
        verbose_name='상태',
    )
    PRIORITY_CHOICES = [
        (1, '낮음'),
        (2, '보통'),
        (3, '높음'),
    ]
    priority = models.PositiveIntegerField(
        choices=PRIORITY_CHOICES,
        default=2,
        help_text='우선순위를 선택해 주세요 (1=낮음, 2=보통, 3=높음).',
        verbose_name='우선순위',
    )

```

### PositiveIntegerField
- 0을 포함한 양의 정수만 저장할 수 있도록 강제하는 django 모델 필드

### blank vs. null
- blank=True: form에서 빈 값 허용
- null=True: db컬럼에서 null 허용
- 문자열 필드에는 blank=True로 충분
    - 사용자가 아무것도 입력하지 않았다는 것을 표현하는 데에는 빈 문자열 하나만으로 충분

## Django Form 심화
### 다양한 Form 살펴보기
1. 기본 Form
```py
class BaseForm(forms.Form):
    title = form.CharField(max_length=50)
```
2. ModelForm
```py
class ProductForm(forms.ModelForm):
    class Meta:
        model = Product
        fields = ['title']
```
3. Widget을 활용한 Form
```py
class WidgetForm(forms.Form):
    username = form.CharField(
        widget=forms.TextInput(
            attrs={
                'class': 'form-control'
            }
        )
    )
```

### MultipleChoiceField
- Form에서 여러 개의 선택지를 동시에 선택할 수 있도록 해주는 필드
```py
class MyForm(forms.Form):
    FAVORITE_COLORS = [
        ('red', 'Red'),
        ('green', 'Green'),
        ('blue', 'Blue'),
    ]
    colors = forms.MultipleChoiceField(
        choices=FAVORITE_COLORS,
        required=False,
        help_text='하나 이상의 색상을 선택하세요',
        widget=forms.CheckboxSelectMultiple,
    )
```

### cleaned_data
- Django 폼에서 유효성 검사(is_valid())를 성공적으로 통과한 "깨끗한" 데이터를 담고 있는 파이썬 딕셔너리
- request.POST: 검증되지 않음, 문자열 형태
- form.is_valid(): 유효성 검사
- form.cleaned_data: 검증완료, 올바른 파이썬 타입으로 변환
