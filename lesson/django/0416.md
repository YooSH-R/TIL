# Django: Form
## Django Form
- HTML form은 비정상적 혹은 악의적 요청 필터링 불가
- 유효성 검사: 수집한 데이터가 정확하고 유효한지 확인하는 과정

### Form class
- Django Form: 사용자 입력 데이터 수집, 처리 및 유효성 검사 수행
### Widgets
- HTML 'input' element의 표현을 담당
- 각 필드가 HTMLl에서 어떻게 렌더링 될지 결정
<br><br>

## Django ModelForm
- 사용자 입력 데이터를 DB에 저장해야 할 때
- Model과 연결된 Form을 자동으로 생성해주는 기능을 제공
### Meta class
- ModelForm의 정보를 작성하는 곳
- 어떤 모델과 연결할지, 어떤 필드를 사용할지 등을 정의하는 설정 공간
### ModelForm 적용
- is_valid()
### save 메서드
- DB 객체를 만들고 저장하는 ModelForm의 인스턴스 메서드
- instance 여부를 통해 생성할지, 수정할 지를 결정
