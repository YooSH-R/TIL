# import
- import matlotlib.pyplot as plt

# 시각화의 기본 요소
- plt.figure(): 도화지 꺼내기(생략 가능)
- plt.plot(x, y, label="범례 이름"): 선 긋기
  - x = [1, 2, 3], y = [1, 2, 3]
- plt.title("제목"): 제목 달기
- plt.xlabel("x축 이름")
- plt.ylabel("y축 이름")
- plt.legent(loc='upper left): 범례 표시
- plt.show(): 결과 확인

# 선 그래프
- 시간에 따른 데이터의 변화(추세), ex) 주식 가격, 기온 변화, 매출 추이
- plt.plot(x, y, color='red', linestyle='--', marker='o'): 선 색깔, 모양, 마커 등을 바꿀 수 있음

# 막대 그래프
- 여러 항복의 크기(양) 비교, ex) 우리 반 학생들의 키, 국가별 GDP, 부서별 매출액
- plt.bar(x, y): 보통 막대 그래프
- plt.barh(x, y): 수평 막대 그래프

# 산점도
- 두 변수 사이의 관계(상관관계)를 볼 때 사용, ex) 공부 시간과 성적, 키와 몸무게, 광고비와 매출
- plt.scatter(x, y, s=점크기, alpha=투명도)
