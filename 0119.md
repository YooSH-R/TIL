## 변수와 메모리
- 객체(object) : 값 + 타입 + 주소 정보를 묶은 것
- a가 리스트 일 때 a = b vs. a = b[:] (얕은 복사)
- a = [1, 2, [3, 4]] 일 때 b = a[:] a[2][0] = 6 print(b[2][0]) = ?
  - 6 (얕은 복사 시 내부의 주소가 그대로 복사)

## 부동 소수점 오차
 - 2진법을 사용하는 컴퓨터는 소수(ex. 0.1) 중 일부를 2진수로 바꾸면 무한 소수가 됨

## Immutable vs. mutable
- Immutable: 문자형, 숫자형, set, tuple
- mutable: list, dictionary

## GUI vs CLI
- GUI: Graphic User Interface
- CLI: Command Line Interface
  - touch: 파일 생성
  - mkdir: 새 디렉토리 생성
  - ls: 현재 작업 중인 디렉토리 내부의 폴더 / 파일 목록 출력
  - rm: 파일 삭제(디렉토리 삭제는 -r 옵션 추가 사용)
  - pwd: 현재 작업 중인 디렉토리 절대 경로 출력

## Git
- 분산 버전 관리 시스템
  - 3가지 영역: Working Directory, Staging Area, Repository
  - Commit: 버전
- Git 동작
  - git init: 로컬 저장소 설정
  - git add: 변경사항이 있는 파일을 staging area에 추가
  - git commit: staging area에 있는 파일들을 저장소에 기록
  - git config --global user.email "메일주소"
  - git config --global user.name "유저네임": commit 작성자 정보 설정
  - git status: 현재 로컬 저장소의 파일 상태 보기
  - git log: commit history 보기
  - git log --oneline: commit 목록 한 줄로 보기
  - git config --global -l: git global 설정 정보 보기
