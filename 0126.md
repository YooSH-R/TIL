## 메서드
- 객체에 속한 함수
    - 객체: 특정 데이터와 그 데이터를 처리하는 기능(메서드)을 하나로 묶은 것
- 클래스(class) 내부에 정의되는 함수
- help함수를 통해 str을 호출해보면 class였다는 것을 확인 가능

## 공통 시퀀스 메서드
- s.index(x)
    - 시퀀스에서 첫 번째로 일치하는 항목 x의 인덱스 반환.
    - 없으면 ValueError 발생

- s.count(x)
    - 시퀀스 s에서 등장하는 항목 x의 개수 반환

## 불변 시퀀스 메서드(문자열 전용)
- 탐색 및 검증 메서드
    - s.find(x)
        - x의 첫 번째 위치(인덱스)를 반환. 없으면 -1을 반환
    - s.isupper()
        - 문자열 내의 모든 케이스 문자가 대문자인지 확인
    - s.islower()
        - 문자열 내의 모든 케이스 문자가 소문자인지 확인
    - s.isalpha()
        - 문자열의 모든 문자가 알파벳이면 True를 반환
        - \* 단순 알파벳이 아닌 유니코드 상 Letter(한국어 포함)

- 조작 메서드(새로운 문자열 반환(불변) -> 재할당 x)
    - s.replace(old, new[, count])
        - old를 new로 count(생략가능) 값(개수) 만큼 바꿈
    - s.strip([chars])
        - 선행과 후행 문자 제거
        - chars 인자는 제거할 문자 **집합**, 기본값은 공백 제거
    - s.split(sep=None, maxsplit=-1)
        - 문자열에 있는 단어들의 리스트를 반환
        - maxsplit이 주어지면 그만큼의 분할 수행, 기본값(-1)은 무제한
        - 공백 기준으로 분리, 앞뒤 공백은 무시
    - s.join(iterable)
        - 구분자(s)로 iterable의 문자열을 연결
        - iterable 사이사이에 s를 붙인 string을 반환
    - s.capitalize()
    - s.title()
    - s.upper()
    - s.lower()
    - s.swapcase()

## 가변 시퀀스 메서드(리스트 전용)
- 값 추가 및 삭제 메서드(i = index, x = value)
    - L.append(x)
    - L.expend(iterable)
    - L.insert(i, x)
    - L.remove(x)
    - L.pop(i)
    - L.clear()

- 정렬 메서드
    - L.reverse()
    - L.sort()
        - L.sort(reverse=True) # 내림차순 정렬
        - .sort()는 리스트의 메서드고 sorted()는 내장 함수
            - sort는 정렬만, sorted는 새로운 리스트 반환, 인자 다양

## 깊은 복사 vs 얕은 복사
- 값은 메모리 주소에 할당, 변수 할당하면 메모리 주소를 가리킴

- 객체 = 값 + 타입 + 주소

- 복사: 원본과 값은 같지만 수정 시 원본에 영향을 주면 안됨
    - a = b 는 b가 a의 주소까지 참조하므로 b라는 새로운 객체가 생성되는 것이 아니기 때문에 복사가 아니다

- 얕은 복사
    - 새로운 객체를 반환
    - 새로운 객체의 요소들은 원본의 요소들이 참조하는 주소를 그대로 참조
        - immutable 객체는 값을 변경할 수 없어 변경 시 새 객체가 생성되므로 문제 없음
        - mutable 객체는 값만 변경 가능하여 원본에 영향을 줄 수 있음

- 깊은 복사
    - 새로운 객체를 반환
    - 새로운 객체의 요소들은 원본과 다른 새로운 주소를 참조(immutable 객체는 주소가 같을 수 있음)
