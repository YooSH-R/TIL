# 데이터와 학습의 이해
## 데이터 구성요소
- Feature(피처, 특성)
  - 모델이 예측에 사용하는 입력정보
- Label(라벨, 목표값)
  - 모델이 예측하려는 정답

# 지도학습의 개념
## 지도학습(Supervised learning)이란?
- 정답 라벨이 있는 훈련 데이터를 사용해 예측 모델을 학습하는 방법

## 지도학습의 종류(회귀 vs 분류)
- 회귀(Regression)
  - 예측하고 싶은 결과값이 숫자일 때 사용
- 분류(Classification)
  - 예측하고 싶은 결과값이 범주일 때 사용

## 지도학습 용어
- 특성(Feature, x)
- 라벨(Label, y)
- 예측값(^y): 학습 과정에서 모델이 내놓은 결과
- 오류(Error): 예측값과 라벨의 차이

# 회귀(Regression)
## 회귀 문제
- 회귀: 입력(feature)으로부터 숫자(output)를 얼마나 정확히 예측할까?

## 회귀 오류: 평균제곱오차(MSE)
- 예측이 얼마나 정확한지 어떻게 알 수 있을까?
  - 모델의 예측 정확도를 숫자로 측정하는 지표 필요
  - 평균제곱오차
    - 각 데이터에서 정답(yi)과 예측(^yi)의 평균 제곱 차이값

## 회귀 설명력: R^2 (결정계수)
- 결정계수
  - Label(y)의 분산 중에서 Feature로 설명되는 비율
  - 평균만 쓰는 단순한 예측보다 모델이 실제값을 얼마나 더 잘 맞추는지를 0~1 사이 값으로 나타냄
  - 1에 가까울수록 설명력이 높고, 낮을수록 설명력이 낮음

# 분류(Classification)
## 분류 문제
- 분류
  - 입력 특성으로부터 출력이 범주값(카테고리)으로 나타내는 예측 문제

## 분류 정확도(Accuracy)
- 분류 모델의 예측 정확도는 어떻게 평가할까?
  - 분류모델은 Label이 범주로 돼 있기 때문에 MSE나 R^2를 사용하기 어렵다.
- 정확도
  - 전체 예측 중에서 올바르게 맞춘 예측의 비율

## 혼동행렬(Confusion Matrix)
- 예측과 실제 값 사이의 관계를 행렬 형태로 표현
- 정밀도(Precision)
  -  양성이라 판정한 것 중 진짜 양성의 비율
- 재현율(Sensitivity or Recall)
  - 진짜 양성 가운데 잡아낸 예측 양성 비율

## 비지도 학습
- 레이블 없이 데이터의 구조&middot;패턴&middot;집단(잠재 서브그룹)을 찾아내는 학습
- 대표 과제: 군집화(clustering), 차원축소(PCA 등), 밀도추정/이상치 탐지
- 출력: 정답 예측이 아니라 구조/요약/표현(embedding)


## 클러스터링
- K-means 클러스터링: K(클러스터 수)를 미리 정해 분할
- 계층적 군집(Hierarchical): K를 사전에 고정하지 않음
